# Task 1: Data Immersion and Wrangling

## ApexPlanet Data Analytics Internship

### Project Overview
This project focuses on Data Immersion and Data Wrangling using the Netflix dataset. The objective was to understand the dataset, clean inconsistent data, handle missing values, and prepare the data for analysis and visualization.

## Dataset
- Dataset: Netflix Movies and TV Shows
- Format: CSV
- Tool Used: Jupyter Notebook (Python)

## Objectives
- Understand the dataset structure.
- Explore the available features.
- Identify missing and duplicate values.
- Clean and preprocess the data.
- Prepare the dataset for further analysis.

## Data Wrangling Steps
- Imported required Python libraries.
- Loaded the Netflix dataset.
- Explored dataset information and summary statistics.
- Checked for missing values.
- Removed duplicate records.
- Handled null values.
- Corrected data types.
- Cleaned unnecessary columns.
- Saved the cleaned dataset as `netflix_cleaned.csv`.

## Tools & Libraries
- Python
- Jupyter Notebook
- Pandas
- NumPy

## Project Outcome
Successfully transformed the raw Netflix dataset into a clean and structured dataset ready for exploratory data analysis and dashboard creation.
